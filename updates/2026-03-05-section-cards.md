# 2026-03-05 - 공고 섹션 카드 (리멤버 스타일)

## 변경 사항

### 1. 상세 페이지 공고 내용 렌더링 개선

- **섹션별 카드 분리**
  - `공고소개`, `주요업무`, `자격요건`, `우대사항`, `복지`, `채용절차`, `기타안내`, `일과 건강`, `음식과 건강`, `성장`, `가족`, `생활 복지` 등 키워드 기준으로 섹션 분리
  - 각 섹션을 **카드** 형태로 렌더링 (테두리, 배경, 패딩)
  - 리멤버 공고처럼 긴 텍스트를 섹션 단위로 나눠 가독성 향상

### 2. CSS 추가

- `.jd-sections`: 섹션 컨테이너 (스크롤 가능)
- `.jd-section-card`: 개별 섹션 카드
- `.jd-section-title`: 섹션 제목
- `.jd-section-body`, `.jd-section-para`, `.jd-section-bullet`: 본문 스타일
