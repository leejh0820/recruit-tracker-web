# 2026-03-09 - 중복 경고, 다크/라이트 모드

## 변경 사항

### 1. 중복 경고 기능

- **입력 시 실시간 경고**: 회사·직무를 입력하면 이미 등록된 지원이 있으면 노란 배너로 표시
- **저장 시 확인**: 같은 회사·직무로 저장 시 `confirm` 다이얼로그로 "그래도 저장할까요?" 확인
- 대소문자·공백 무시하고 비교

### 2. 다크/라이트 모드

- **헤더 토글 버튼**: 🌙 / ☀️ 클릭으로 테마 전환
- **localStorage 저장**: 선택한 테마 유지
- **시스템 설정 반영**: 첫 방문 시 `prefers-color-scheme` 적용
- **플래시 방지**: `beforeInteractive` 스크립트로 초기 테마 즉시 적용
- 카드, 입력, 테이블, 버튼 등 전체 UI 다크 모드 스타일

### 3. UI

- `.duplicate-warning` 스타일 추가 (앰버 배경, 테두리)
- `--company-accent` CSS 변수 (라이트/다크 각각)
