# 2026-03-05 - 지원 수정/삭제

## 변경 사항

### 1. 백엔드 API

- **PATCH /applications/{id}**
  - 지원 내용 일부 또는 전체 수정
  - `ApplicationUpdate` 스키마 (모든 필드 optional)
- **DELETE /applications/{id}**
  - 지원 삭제
  - 204 또는 `{"ok": true}` 반환

### 2. 상세 페이지 UI

- **수정 버튼**
  - 클릭 시 인라인 편집 폼 표시
  - 날짜, 출처, 회사, 직무, 위치, 근무형태, 연봉, 지원여부, 단계, URL, 공고 내용/메모 수정 가능
  - 저장 시 PATCH 호출 후 화면 갱신
  - 취소 시 편집 모드 해제
- **삭제 버튼**
  - 클릭 시 확인 다이얼로그
  - 확인 시 DELETE 호출 후 목록 페이지(`/`)로 이동
