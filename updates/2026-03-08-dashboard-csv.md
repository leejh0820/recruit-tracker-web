# 2026-03-08 - 대시보드, CSV 내보내기, 연결 안정화

## 변경 사항

### 1. CSV 내보내기

- 지원 리스트 툴바에 **CSV 내보내기** 버튼 추가
- 검색/필터/정렬 적용된 목록 기준으로 다운로드
- 파일명: `recruit-tracker-YYYY-MM-DD.csv`
- UTF-8 BOM 포함 (엑셀 한글 깨짐 방지)
- 컬럼: 날짜, 출처, 회사, 직무/포지션, 위치, 근무형태, 연봉/페이, 단계, 지원완료

### 2. 대시보드/통계

- **단계별 지원 수**: 관심, 지원완료, OA, 1차면접, 2차면접, 오퍼, 리젝트 막대 차트
- **월별 지원 수**: 월별 지원 건수 막대 차트
- 지원 1건 이상일 때만 대시보드 표시
- CSS 기반 막대 차트 (외부 라이브러리 없음)

### 3. 연결 오류 개선

- 백엔드 연결 실패 시 안내 메시지: "백엔드 서버에 연결할 수 없습니다..."
- **다시 시도** 버튼으로 재요청 가능
- 실행 명령어 안내 추가

### 4. 프론트엔드 안정화

- **Google 폰트(Geist) 제거**: 빌드 시 네트워크 요청으로 멈춤 방지
- **React Compiler 비활성화**: dev 서버 느려짐 완화
- `run.sh` 스크립트 추가 (백엔드/프론트엔드 분리 실행)
